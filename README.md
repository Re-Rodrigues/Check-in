# 📱 Check-in

Projeto desenvolvido para a disciplina de Desenvolvimento Móvel Avançado

## 🚀 Como executar

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   cd seu-repositorio
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o projeto:
   ```bash
   npx expo start
   ```

4. Escolha como rodar:
   - **Android:** pressione `a`
   - **iOS:** pressione `i`
   - **Dispositivo físico:** escaneie o QR code com o app Expo Go
   - Recomendado: Caso o aplicativo do expo não esteja reconhecendo o wifi, utilize "npx expo start --tunnel"
